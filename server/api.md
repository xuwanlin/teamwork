## 获取图片列表
- api/imglink?file=test1&count=2
- 请求方式：get
- 参数：
  - file（字段、文件名）
  - count（字段：条数，返回全部可以不写）
求
- json文件放置目录在mock/imgLink

成功
```
{
    "code": 0,
    "success": "获取数据成功！",
    "list": [
        {
            "src": "http://localhost:3000/imgLink/test2/1.jpg",
            "link": "#1"
        },
        {
            "src": "http://localhost:3000/imgLink/test2/2.jpg",
            "link": "#2"
        }
    ]
}
```
失败
```
{
    "code": 1,
    "error": "文件路径错误！"
}
```
## 注册
  - /api/reg
- post请求
- 参数：
  - username（用户名字段）
  - password（密码字段）

用户名重复
```
{
    "code": 1,
    "error": "用户名已存在，请重新输入！"
}
```
注册成功
```
{
    "code": 0,
    "success": "注册成功",
    "user": {
        "username": "xuwanlin111"
    }
}
```
